Installation

  • Downloaded this release to $HOME/Sites/http2
  • Renamed it 
    mv simplehttp2server_darwin_amd64 simplehttp2server
  • Next, I did a 
    chmod +x simplehttp2server
  • Move to /usr/local/bin 
    mv simplehttp2server /usr/local/bin/simplehttp2server
  • Restart terminal (or open new tab)

Learnings

  1. HTTP/2 only works over TLS (HTTPS)
  2. Enable Protocol column in network tab to see "h2"
  3. Don't use "Disable cache" in DEV Tools
  4. "Link:" headers is NOT how you do push
